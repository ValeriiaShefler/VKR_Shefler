       ЦИФРОВАЯ МОДЕЛЬ МЕЖДУНАРОДНОЙ СТАНДАРТНОЙ АТМОСФЕРЫ (МСА)
            С ИИ-АППРОКСИМАЦИЕЙ ПАРАМЕТРОВ

1. НАЗНАЧЕНИЕ ПРОГРАММЫ И ОБЛАСТЬ ПРИМЕНЕНИЯ
--------------------------------------------------------------------------------
Программа предназначена для получения параметров Международной стандартной
атмосферы (МСА) в диапазоне высот от 0 до 20000 м в двух режимах:
  • Эталонный расчёт по ГОСТ 4401-81 / ISO 2533:1975.
  • ИИ-аппроксимация (нейросетевая модель).

МСА используется в следующих задачах авиационной и аэрокосмической отрасли:
  • Проектирование летательных аппаратов (аэродинамика, прочность).
  • Расчёт лётно-технических характеристик (дальность, скороподъёмность).
  • Сертификация авиационных двигателей и бортового оборудования.
  • Калибровка высотомеров и навигационных систем.
  • Лётные испытания и обработка полётных данных.
  • Численное моделирование (CFD, CAE) в цифровых двойниках.


2. ОСНОВНЫЕ ПАРАМЕТРЫ МСА И ИХ ВЛИЯНИЕ НА ПОЛЁТ
--------------------------------------------------------------------------------
2.1. Температура воздуха (T, K)
  • Влияет на скорость звука:              a = √(γ * R * T)
    где γ = 1.4 (показатель адиабаты), R = 287.053 Дж/(кг·К).
  • От температуры зависит число Маха:      M = V / a
  • Влияет на тягу воздушно-реактивного двигателя.
  • Определяет высоту тропопаузы (11 км), где T становится постоянной.

2.2. Давление (P, Па)
  • Барометрическая формула:               P(h) = P₀ * exp(-Mgh / (RT))
  • Используется для расчёта высоты по давлению (эшелонирование).
  • Влияет на показания анероидных высотомеров.

2.3. Плотность (ρ, кг/м³)
  • Подъёмная сила крыла:                 Y = C_y * (ρ * V² / 2) * S
  • Лобовое сопротивление:                X = C_x * (ρ * V² / 2) * S
  • Тяга двигателя (взлётная, крейсерская) пропорциональна ρ.
  • При подъёме на высоту плотность падает → подъёмная сила и тяга падают.

2.4. Скорость звука (a, м/с)
  • Определяет число Маха:                 M = V / a
  • Влияет на волновое сопротивление при трансзвуковых скоростях.
  • Важна для расчёта критической скорости (Mкр).


3. СТРУКТУРА МСА В ДИАПАЗОНЕ 0–20000 М
--------------------------------------------------------------------------------
МСА делится на два слоя:

  • Тропосфера (0–11000 м):
    Температура падает линейно:      T = 288.15 – 0.0065 * h
    Давление и плотность уменьшаются по экспоненте.

  • Нижняя стратосфера (11000–20000 м):
    Температура постоянна:            T = 216.65 K = const
    Давление и плотность продолжают падать по экспоненте.

На высоте 11000 м (тропопауза) происходит излом температурного профиля,
что хорошо видно на графике температуры на вкладке «Калькулятор МСА».


4. ИСПОЛЬЗОВАННЫЕ НОРМАТИВНЫЕ ДОКУМЕНТЫ
--------------------------------------------------------------------------------
  • ГОСТ 4401-81   «Атмосфера стандартная. Параметры»
  • ISO 2533:1975  «Standard Atmosphere»
  • ICAO Standard Atmosphere (Документ 7488)
  • ГОСТ Р 57700.37-2021 «Цифровые двойники изделий. Общие положения»


5. АЛГОРИТМ РАСЧЁТА ЭТАЛОННЫХ ПАРАМЕТРОВ
--------------------------------------------------------------------------------
Программа использует кусочно-аналитический метод:

  • Определяется слой атмосферы по заданной высоте.
  • Если слой тропосферы:   T(h) = T₀ - β·h (β = 0.0065 K/м).
  • Если слой стратосферы:  T = 216.65 K.
  • Давление вычисляется интегрированием гидростатического уравнения
    с учётом переменной или постоянной температуры.
  • Плотность:              ρ = P / (R * T)
  • Скорость звука:         a = √(γ * R * T)

Формулы соответствуют [ГОСТ 4401-81, приложение 1].


6. ИИ-АППРОКСИМАЦИЯ (НЕЙРОСЕТЕВАЯ МОДЕЛЬ)
--------------------------------------------------------------------------------
6.1. Применяемая архитектура
  • Многослойный перцептрон (MLPRegressor из scikit-learn).
  • Вход: высота h (м) → после нормализации и полиномиальных признаков.
  • Скрытые слои: 50 и 25 нейронов, активация ReLU.
  • Выход: T, P, ρ, a (отдельные модели для каждого параметра).

6.2. Обучение
  • Данные: эталонные значения МСА (высоты 0..20000 м, шаг 100 м).
  • Разделение: 80% обучение, 20% тест.
  • Оптимизатор: Adam, итераций: до 3000.

6.3. Верификация и валидация
  • Метрики: MAE (средняя абсолютная ошибка), RMSE (среднеквадратичная),
    R² (коэффициент детерминации).
  • Требования к цифровым моделям — согласно ГОСТ Р 57700.37-2021.

6.4. Полученные метрики (на тестовой выборке)
  • T:   R² ≈ 0.91,   MAE ≈ 5.7 K
  • P:   R² ≈ 0.99,   MAE ≈ 1783 Pa
  • ρ:   R² ≈ 0.99,   MAE ≈ 0.028 kg/m³
  • a:   R² ≈ 0.75,   MAE ≈ 6.0 m/s

Высокая точность по P и ρ делает модель пригодной для инженерных расчётов.


7. ЭКСПОРТ ДАННЫХ ДЛЯ ПРОИЗВОДСТВЕННЫХ СИСТЕМ
--------------------------------------------------------------------------------
Программа позволяет выгрузить полный профиль атмосферы (0–20000 м, шаг 100 м)
в форматах:

  • CSV  — для Excel, MATLAB, Python (pandas).
  • JSON — для веб-API, микросервисов.
  • XML  — для обмена с унаследованными системами (PLM, PDM).
  • TXT  — для чтения любым текстовым редактором, простых протоколов.

Файлы могут быть переданы в SCADA-системы, на ПЛК через OPC UA или
использованы в цифровых двойниках.


8. ИНСТРУКЦИЯ ПО ЗАПУСКУ
--------------------------------------------------------------------------------
Программа портативная — не требует установки Python.

  1. Распакуйте архив в любую папку.
  2. Запустите файл run_app.bat (двойным кликом).
  3. Откроется консоль, а через 3–5 секунд — браузер с адресом:
     http://localhost:8000
  4. Для остановки сервера закройте консоль или нажмите Ctrl+C.


9. ИНТЕРФЕЙС И РАБОТА С ПРОГРАММОЙ
--------------------------------------------------------------------------------
9.1. Вкладка «Калькулятор МСА» (эталон)
  • Введите высоту в метрах (0–20000), нажмите «Рассчитать».
  • Появятся T, P, ρ, a по ГОСТ/ISO.
  • График температуры отображает профиль от 0 до 20000 м.
  • Кнопка «Экспорт текущих данных» сохраняет параметры текущей высоты.

9.2. Вкладка «ИИ-аппроксимация»
  • Кнопка «Обучить нейросеть» — запускает обучение (10–30 секунд,
    только при первом запуске или после очистки папки saved_models).
  • Дождитесь сообщения «Обучение успешно завершено».
  • Введите высоту, нажмите «Предсказать» — нейросеть выдаст параметры.
  • Результаты предсказания можно сравнить с эталоном на соседней вкладке.

9.3. Вкладка «Верификация и валидация»
  • Нажмите «Получить метрики» — появится таблица с MAE, RMSE, R² для T, P, ρ, a.
  • R² > 0.98 для P и ρ означает отличное качество модели.
  • Метрики вычисляются на тестовой выборке (20% данных).

9.4. Вкладка «Экспорт»
  • Кнопки CSV / JSON / XML / TXT — скачивают полный профиль МСА (201 строка).
  • Файлы могут быть использованы для дальнейшей обработки в CAE-системах.

12. НАУЧНАЯ НОВИЗНА И ПРАКТИЧЕСКАЯ ЗНАЧИМОСТЬ
--------------------------------------------------------------------------------
Научная новизна:
  • Разработана архитектура цифрового двойника МСА с поддержкой
    нейросетевой аппроксимации.
  • Предложен метод верификации и валидации ИИ-модели в соответствии
    с ГОСТ Р 57700.37-2021.

Практическая значимость:
  • Готовый программный модуль может быть встроен в PLM/ERP системы.
  • Экспорт в CSV/JSON/XML/TXT позволяет передавать данные на ПЛК, в SCADA,
    CAE-комплексы (ANSYS, OpenFOAM, Simcenter).
  • Портативность обеспечивает запуск на любом компьютере без установки.
