# Цифровая модель Международной стандартной атмосферы (МСА) с ИИ-аппроксимацией параметров

## 1. Назначение программы и область применения

Программа предназначена для получения параметров Международной стандартной атмосферы (МСА) в диапазоне высот от 0 до 20000 м в двух режимах:

- **Эталонный расчёт** по ГОСТ 4401-81 / ISO 2533:1975.
- **ИИ-аппроксимация** (нейросетевая модель).

**Область применения в авиации и аэрокосмической отрасли:**

- Проектирование летательных аппаратов (аэродинамика, прочность)
- Расчёт лётно-технических характеристик (дальность, скороподъёмность)
- Сертификация авиационных двигателей и бортового оборудования
- Калибровка высотомеров и навигационных систем
- Лётные испытания и обработка полётных данных
- Численное моделирование (CFD, CAE) в цифровых двойниках

---

## 2. Основные параметры МСА и их влияние на полёт

### 2.1. Температура воздуха (T, K)

- Влияет на скорость звука:  
  `a = √(γ * R * T)`,  
  где γ = 1.4 (показатель адиабаты), R = 287.053 Дж/(кг·К)
- От температуры зависит число Маха: `M = V / a`
- Влияет на тягу воздушно-реактивного двигателя
- Определяет высоту тропопаузы (11 км), где температура становится постоянной

### 2.2. Давление (P, Па)

- Барометрическая формула: `P(h) = P₀ * exp(-Mgh / (RT))`
- Используется для расчёта высоты по давлению (эшелонирование)
- Влияет на показания анероидных высотомеров

### 2.3. Плотность (ρ, kg/m³)

- Подъёмная сила крыла: `Y = C_y * (ρ * V² / 2) * S`
- Лобовое сопротивление: `X = C_x * (ρ * V² / 2) * S`
- Тяга двигателя (взлётная, крейсерская) пропорциональна ρ
- При подъёме на высоту плотность падает → подъёмная сила и тяга падают

### 2.4. Скорость звука (a, m/s)

- Определяет число Маха: `M = V / a`
- Влияет на волновое сопротивление при трансзвуковых скоростях
- Важна для расчёта критической скорости (Mкр)

## 3. Структура МСА в диапазоне 0–20000 м

МСА делится на два слоя:

| Слой | Высоты | Закон изменения температуры |
|------|--------|------------------------------|
| Тропосфера | 0–11000 м | `T = 288.15 – 0.0065 * h` (линейное падение) |
| Нижняя стратосфера | 11000–20000 м | `T = 216.65 K = const` |

На высоте **11000 м (тропопауза)** происходит излом температурного профиля, что хорошо видно на графике температуры на вкладке «Калькулятор МСА».

## 4. Использованные нормативные документы

- ГОСТ 4401-81 «Атмосфера стандартная. Параметры»
- ISO 2533:1975 «Standard Atmosphere»
- ICAO Standard Atmosphere (Документ 7488)
- ГОСТ Р 57700.37-2021 «Цифровые двойники изделий. Общие положения»

## 5. Алгоритм расчёта эталонных параметров

Программа использует кусочно-аналитический метод:

1. Определяется слой атмосферы по заданной высоте
2. Если слой тропосферы: `T(h) = T₀ - β·h` (β = 0.0065 K/м)
3. Если слой стратосферы: `T = 216.65 K`
4. Давление вычисляется интегрированием гидростатического уравнения
5. Плотность: `ρ = P / (R * T)`
6. Скорость звука: `a = √(γ * R * T)`

Формулы соответствуют [ГОСТ 4401-81, приложение 1].

## 6. ИИ-аппроксимация (нейросетевая модель)

### 6.1. Архитектура

- **Тип модели:** Многослойный перцептрон (MLPRegressor из scikit‑learn)
- **Вход:** высота `h` (м) → после нормализации и полиномиальных признаков
- **Скрытые слои:** 50 и 25 нейронов, активация ReLU
- **Выход:** отдельные модели для T, P, ρ, a

### 6.2. Обучение

- **Данные:** эталонные значения МСА (высоты 0..20000 м, шаг 100 м)
- **Разделение:** 80% обучение, 20% тест
- **Оптимизатор:** Adam, итераций: до 3000

### 6.3. Верификация и валидация

- **Метрики:** MAE (средняя абсолютная ошибка), RMSE (среднеквадратичная), R² (коэффициент детерминации)
- **Нормативная база:** ГОСТ Р 57700.37-2021

### 6.4. Полученные метрики (на тестовой выборке)

| Параметр | R²    | MAE           |
|----------|-------|---------------|
| T        | 0.91  | 5.7 K         |
| P        | 0.99  | 1783 Pa       |
| ρ        | 0.99  | 0.028 kg/m³   |
| a        | 0.75  | 6.0 m/s       |

Высокая точность по **P** и **ρ** делает модель пригодной для инженерных расчётов.

## 7. Экспорт данных для производственных систем

Программа позволяет выгрузить полный профиль атмосферы (0–20000 м, шаг 100 м) в форматах:

| Формат | Назначение |
|--------|-------------|
| CSV    | Excel, MATLAB, Python (pandas) |
| JSON   | Веб-API, микросервисы |
| XML    | Обмен с унаследованными системами (PLM, PDM) |
| TXT    | Чтение любым текстовым редактором, простые протоколы |

Файлы могут быть переданы в SCADA-системы, на ПЛК через OPC UA или использованы в цифровых двойниках.

## 8. Инструкция по запуску

Программа **портативная** — не требует установки Python.

1. Распакуйте архив в любую папку
2. Запустите файл `run_app.bat` (двойным кликом)
3. Откроется консоль, а через 3–5 секунд — браузер с адресом:  
   `http://localhost:8000`
4. Для остановки сервера закройте консоль или нажмите **Ctrl+C**

## 9. Интерфейс и работа с программой

### 9.1. Вкладка «Калькулятор МСА» (эталон)

- Введите высоту (0–20000 м), нажмите **«Рассчитать»**
- Появятся `T, P, ρ, a` по ГОСТ/ISO
- График температуры отображает профиль от 0 до 20000 м
- **«Экспорт текущих данных»** — сохраняет параметры текущей высоты

### 9.2. Вкладка «ИИ-аппроксимация»

- **«Обучить нейросеть»** — запускает обучение (10–30 секунд, только при первом запуске)
- Дождитесь сообщения **«Обучение успешно завершено»**
- Введите высоту, нажмите **«Предсказать»** — нейросеть выдаст параметры

### 9.3. Вкладка «Верификация и валидация»

- **«Получить метрики»** — таблица с MAE, RMSE, R² для T, P, ρ, a
- `R² > 0.98` для P и ρ означает отличное качество модели

### 9.4. Вкладка «Экспорт»

- Кнопки **CSV / JSON / XML / TXT** — скачивают полный профиль МСА (201 строка)

## 11. Устранение возможных проблем

| Проблема | Решение |
|----------|---------|
| Ошибка «Модель не обучена» | Вкладка ИИ → **«Обучить нейросеть»** |
| Порт 8000 уже занят | Закрыть все окна с программой, перезагрузить ПК |
| Браузер не открывается | Вручную открыть `http://localhost:8000` |
| XML не скачивается | Использовать прямую ссылку: `http://localhost:8000/export/full_profile?format=xml` |
